# file_transmitting_application
100% java25 application to send a file in an unsafe netwrk.
порт захардкожен по соображениям стиля.

клиентская версия не может запуститься из джарника на windows 10 по причине java fx. с помощью батника -- может.
батник лежит в архиве дистрибуции в папке бин.

джарник серверной версии лежит в \дистрибуции
серверная версия может потребовать права суперпользователя на некоторых системах из-за порт<1024

проводились тесты по отправке файлов в локальной сети и некому "локалхосту". в обоих случаях тесты были успешны.
